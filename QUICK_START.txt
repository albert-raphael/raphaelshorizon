## ğŸ› Troubleshooting

### "Failed to fetch" errors
- Check backend is running: `curl http://localhost:5002/api/health`
- Verify ports aren't blocked
- Check CORS settings in backend

### Admin login not working
- Run: `cd backend && node create-admin.js`
- Check credentials: admin@raphaelshorizon.com / admin123

### MongoDB connection issues
- Verify `.env` MONGO_URI is correct
- Check MongoDB Atlas IP whitelist
- Ensure network connectivity
